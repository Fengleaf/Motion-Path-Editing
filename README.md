## 使用
左 / 右側為 MENU，可透過 < > 符號隱藏 Panel。滑鼠左鍵點擊控制點，可以拖動。相機可透過滑鼠中鍵、右鍵、WASD 等方式控制。

### MENU-File
讀取 BVH 檔案 / 存取 路徑檔案。

### MENU-Edit
清出場景、相機控制、播放。

### MENU-List (左邊)
選取場景中不同的 BVH。

### MENU-Edit (左邊)
指定 BVH 的顯示與否、Motion 連接、Motion blend。
